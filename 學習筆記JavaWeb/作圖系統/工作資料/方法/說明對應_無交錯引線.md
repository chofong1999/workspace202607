# 說明對應：無交錯引線方法

- 適用工作：`WRK-20260720-408T2R4T`
- 方法版本：`3`
- 建立原因：使用者指出不同顏色長引線大量交叉，破壞閱讀順序。

## 固定編號

| 編號 | 分類 |
|---|---|
| 1 | HTML |
| 2 | CSS |
| 3 | JSP 指令／JSP 註解 |
| 4 | JSTL |
| 5 | JSP Action |
| 6 | Java scriptlet |
| 7 | EL |
| 8 | JavaScript |

編號只用來建立框選區與說明框的一一對應，不取代固定兩欄八項圖例。

## 配置規則

1. 分類框右側空白處放同色圓形編號；說明框標題使用相同顏色與相同編號。
2. 同一分類有多個分離框時，預設只在最大或最具代表性的外層區塊旁放一個編號；其他同色框或括號不重複放編號。右側只保留一個合併說明框，文字列出涵蓋行號。
3. HTML、CSS、JSP 指令／註解與 JSTL 預設不畫跨畫面的長引線；以「顏色＋編號」完成對應。
4. 多個 EL 仍依 SOP 使用粉紅群組括號；只允許從括號編號到 EL 說明框的短連線。
5. JavaScript 或 scriptlet 若字元外側空白不足以安全放入 2 px 細框，禁止硬畫細框；改用該行右側的同色單行括號與編號。
6. 說明框放在右側固定欄，依上到下閱讀順序排列；不得因分類清單順序而使對應關係反向。
7. 不同顏色連線交叉數必須為 0。
8. 任一連線不得穿過其他顏色的群組括號、分類框、編號、說明框或程式字形。
9. 能用編號對應時不畫線；能用短水平線時不畫長斜線。
10. 若在單張圖仍無法達到零交叉，移動說明框或拆頁，不得接受交錯版面。

## 不同顏色框線分離

1. 對每張圖的所有分類框兩兩檢查，不只檢查使用者指出的單一位置。
2. 不同顏色的平行框線不得共線、重疊或貼合；2 px 框線的中心距至少 4 px，使兩條線的可見邊緣之間至少保留 2 px。
3. 巢狀結構若共用左界、右界、上界或下界，優先把外層框向外擴張，內層框保持貼近自己的實際字形範圍。
4. 相鄰行之間沒有足夠空間同時容納兩條完整框線時，不得把任一框線推入字形，也不得留下缺少一邊的不完整矩形；改將較小或附屬分類改成同色右括號。
5. 括號必須具有清楚的上下短橫線與外側垂直線；與其他不同色框線、括號、編號及短連線的距離至少 4 px。
6. 調整一處後必須重跑整張及整批的框線配對掃描，不能只確認被指出的位置。
7. 框線分離不得改變來源碼、EL 覆色範圍、分類語意或說明內容。

## 編號與標記元件

1. 同一分類每頁預設只保留一個代表編號；巢狀同類框只在最外層或最大範圍放編號。
2. 分離但同類的其他區塊依框線顏色對應同一說明，不因每個框都加編號而增加雜訊。
3. 不同分類編號不可互疊；編號與其他框線、括號、短連線的可見間距至少 4 px。
4. EL、JavaScript、scriptlet 或 JSP 的括號彼此相鄰時，上下端點與其他顏色標記至少間隔 4 px。
5. 每次調整後掃描全部編號矩形、框線段、括號線段及短連線，不以人工只看被指出位置取代。

## QA

- 長距離跨畫面引線數：0。
- 不同顏色連線交叉數：0。
- 編號與說明框對應錯誤數：0。
- 編號遮擋程式字形數：0。
- 說明框與分類框重疊面積：0。
- EL／JavaScript 短連線穿越其他標記數：0。
- 不同色框線共線或重疊組數：0。
- 不同色平行框線中心距不足 4 px 的組數：0。
- 不完整矩形數：0。
- 明確括號與其他不同色標記距離不足 4 px 的組數：0。
- 同一分類同頁重複編號數：0。
- 編號互疊或碰觸其他標記數：0。
- 因分離框線而新增的程式字形碰觸數：0。
