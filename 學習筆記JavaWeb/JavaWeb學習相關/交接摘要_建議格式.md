# JavaWeb作圖工作的交接摘要建議

本文件只提供摘要欄位，不直接修改交接資料。

```text
- 作圖系統位置：
- 作圖系統版本：0.2-test
- 視覺規格版本：
- 外部規格：SRC-SPEC-001／實際SOP版本
- 已確認樣板版本：
- 當前任務：
- 當前候選：
- 是否允許批次：
- 重要阻塞或待使用者決定：
- 下一步：
- 詳細作圖資料入口：
```

同步原則：

1. 先依交接制度完成接手。
2. 日常只在重要里程碑或狀態改變時更新摘要。
3. 工作告一段落時，依交接系統完成正式交接與日誌。
4. 不把完整SOP、逐張QA及候選全文複製進交接資料。
5. 不修改交接制度版本、核心規則或員工制度。
